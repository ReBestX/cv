# CV Repository

This repository contains professional curriculum vitae provided for review and reference purposes.

The documents in this repository are maintained to present up-to-date professional profiles and may be updated periodically without changing their public access links.

## Contents
- Individual CV files in PDF format

## Usage
The files are intended for viewing and reference only.  
They may be shared as part of professional applications, proposals, or evaluation processes.

## Notes
- File names and links are kept stable to ensure consistent access.
- Updates to the documents do not affect the public URLs.
